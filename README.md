Mise en Place

Mise en Place is a Smart Utility Toolkit for everyday use in the kitchen.
It contains a basic Kitchen Timer, Unit Converters for cooking measurements, and a Notes and Tasks section.
Mise en Place is your Kitchen Companion — with everything in its place.

---

Features

- Modern UI built with Jetpack Compose
- Unit Converter with Length, Cooking Measurements, Weight and Currency tools
- A Tasks section
- Notes section to log meals and save recipes
- Kitchen Timer with pause, resume and reset
- Clean, food-forward UI with smooth navigation

---

Tech Stack

- Language: Kotlin
- Framework: Android Jetpack Compose
- Architecture: MVVM
- Tools: Android Studio
- Local Storage: Room (SQLite)
